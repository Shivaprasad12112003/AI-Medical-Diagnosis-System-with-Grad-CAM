# AI Medical Diagnosis System with Grad-CAM

This project detects Pneumonia from chest X-ray images using Deep Learning and Explainable AI.

## Features
- Pneumonia Detection
- ResNet50 Transfer Learning
- Grad-CAM Heatmap Visualization
- Medical Report Generation
- Gradio Web Interface

## Technologies Used
- TensorFlow
- ResNet50
- OpenCV
- Gradio
- Python

## How to Run

### Install Requirements
pip install -r requirements.txt

### Train Model
python train.py

### Run Application
python app.py
